# Sistem Pakar Prediksi Hasil Tanaman Berbasis Fuzzy Logic

## Deskripsi
Sistem ini merupakan sistem pakar berbasis Fuzzy Logic untuk memprediksi hasil tanaman
berdasarkan parameter unsur hara tanah dan lingkungan, yaitu:
- Nitrogen (N)
- Fosfor (P)
- Kalium (K)
- Temperatur
- Kelembaban
- pH
- Curah Hujan

Sistem dikembangkan menggunakan Java untuk proses inferensi dan JavaScript
untuk visualisasi serta justifikasi aturan.
